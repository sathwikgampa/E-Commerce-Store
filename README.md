# Full-Stack E-Commerce Store

A comprehensive E-Commerce web application built with a modern tech stack. This platform allows users to browse a product catalog, manage their shopping carts, and place orders with options for local store pickup and delivery. It also features a dedicated secure login and an admin layout for store management.

## 🚀 Features

- **User Authentication**: Secure user login and registration flows.
- **Product Catalog**: Beautifully designed, responsive catalog for products and detailed product view pages.
- **Shopping Cart**: Fully functional cart system using global state management (Zustand).
- **Checkout Flow**: Simple and intuitive checkout process for end-users.
- **Admin Dashboard**: Dedicated panels for administrators to securely log in and manage the store.
- **Responsive Design**: Mobile-first, fully responsive sleek UI built with Tailwind CSS.
- **Fast Performance**: Powered by Vite and React on the frontend.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Tailwind CSS, Lucide React (Icons), React Router, Zustand (State Management)
- **Backend**: Node.js, Express
- **Styling**: Tailwind CSS & customized CSS configuration for smooth modern aesthetics.

## 📂 Project Structure

- `/frontend` - The React application (built with Vite).
- `/backend` - The Node.js Express server.

## 💻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.
- Git installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sathwikgampa/E-Commerce-Store.git
   ```

2. **Navigate to the frontend directory:**
   ```bash
   cd E-Commerce-Store/frontend
   ```

3. **Install frontend dependencies:**
   ```bash
   npm install
   ```

4. **Start the frontend development server:**
   ```bash
   npm run dev
   ```

5. **Navigate to the backend directory:**
   ```bash
   cd ../backend
   ```

6. **Start the backend server:**
   ```bash
   npm start
   ```
